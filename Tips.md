# Some tips
**Grading System**
- WA: wrong answer
- RTE: run-time error
- MLE: memory limit exceeded
- TLE: time limit exceeded
- AC: all correct

**Big O rule of thumb**  
Big O notation is used as an abstraction of time complexity.  
In general, the value inside the O() should be >= time limit * 1000000  
- 10^6: O(N)/O(log N)/O(1)
- 10^5: O(N log N)
- 10^3: O(N^2)
- 10^2: O(N^3)
- 10: O(N!)/O(2^N)
